    • Physical applications

      Flux allows to model different physical phenomena, and includes various applications (magnetic, electrical, thermal and thermal coupled).

    • Circuit and Kinematic Couplings

      Flux is coupled to a circuit context allowing to model the sources of the devices.Flux also allow to model the movement via a kinematic coupling.

    • Co-simulation between Flux and another software

      Flux is able to couple with several softwares allowing to model mutliphysics phenomena.

    • Dedicated couplings

      Several dedicated couplings are implemented in Flux to, carry out vibro-acoustic studies, magneto-thermal studies and more.

    • 3D Overlay

      Flux has predefined models for rotating machines via Overlays allowing to describe a geometry, a mesh and a physics adapted to the wished machine.

Correction of geometric defects: about

Modes of correction of defects

The correction of defects can be carried out in two ways:

  • automatically: using algorithms of correction / simplification of the geometry
  • manually: using specific tools

Automatic correction

The automatic correction enables to correct the following geometric defects:

  • superimposed points
  • superimposed lines
  • intersection line-line
  • intersection line-face
Note: The automatic correction is planned especially for the 2D geometry, the result in 3D is not guaranteed.

Automatic simplification

The simplification is used to simplify or "reduce" the geometry. The algorithm of automatic simplification removes some lines and points by assembling the lines of the same type.

Only the lines of the following types can be fused:

  • segments located on the tangent of the straight lines
  • arcs of circle having the same angle of curve

Manual correction

The manual correction is used:

  • on the one hand to correct the geometric defects by defect type
  • on the other hand to simplify the geometry manually

Types of fusion

There are 3 reduce types for the command Reduce continuous lines in one line:

  • Automatic
  • Arc
  • Segment

The choice of the reduce type depends on the types of lines. Some precautions concerning the choice of the reduce type and the type of lines to be reduced are necessary to preserve the geometry of the device.
