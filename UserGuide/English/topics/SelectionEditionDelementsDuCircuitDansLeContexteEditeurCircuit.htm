Selection, editing of circuit components in the circuit editor context

Introduction

This section presents the various functions of the electric circuit editor context connected with the selection and editing of the components of the electric circuit.

These manipulations can be done starting from:

  • The Graphic sheet
  • The Data tree
  • The Circuit menu

Graphic selection cursor

In the graphic zone, the selection of circuit components is carried out with the mouse which has this cursor by default.

Selection modes by component

Each component has its own selection mode appropriately associated to an icon to be commanded.

Concerning the components, there is a selection by type, which permits to choose in a selection only components of the same type.

Concerning the connection wires there are different modes of selection. The wires consist of an assembly of segments. It is therefore possible to select:

  • a segment
  • an assembly of segments (connection)
  • an assembly of connections (potential)

The different modes of selections of wires are presented in the table below:

Mode of selection Icon Selection
Segment of one segment
Connection of the assembly of segments between two connecting points or between one connecting point and a component terminal or between two component terminals
Potential of the assembly of elements with the same potential (connections, connecting points)

By default, the active mode of selection is the Segment mode.

Select and Edit an component from the Data tree

To select and then edit a component of the electric circuit starting from the Data tree :

Step Action
1

Select the component in the Data tree :

  • point and click on the desired component
→ The element is highlighted
2

Edit the component in the Data tree :

  • double click on the selected component

or

  • right click on the selected component and choose the Edit or Edit array commands
→

The dialogue box corresponding to the electric component is opened.

The user can characterize or modify the component of the electric circuit

Select and Edit an electric component by graphic sheet

To Select and then edit a component of the electric circuit starting form the Graphic sheet :

Step Action
1

Select the component starting from the graphic sheet:

  • point and click on the desired component
→ The graphic component is shining
2

Edit the component starting from the graphic sheet:

  • double click on the selected component

or

  • click right on the selected component and choose the Edit or Edit array commands
→

The dialogue box corresponding to the electric component is opened.

The user can characterize or modify the component of the electric circuit

Select an assembly of electric components

To select an assembly of components (starting only from the graphic zone):

Step Action
1

In the graphic sheet choose the zone to select:

  • Slide a selection rectangle around the electric components to select (maintain the left button of the mouse pressed)
→ The graphic elements inside the selection zone appear shining
2

To add another zone to the selection

  • Repeat step 1 while maintaining the CTRL key pressed in order to add components to the selection group
→ The user can then consider a manipulation of this selection group (copy/paste, slide, rotate…)

Select an assembly of electric components of the same type

To operate a selection of an assembly of electric components of the same type:

Step Action
1 Activate the by type selection by clicking on one of the modes of selection of component (for example )
→ A dialogue box Type selection is opened
2 Choose the desired type of component
→ The dialogue box must remain open
3

In the graphic sheet choose the zone to select:

  • Slide a selection rectangle around the component to select (maintain the left button of the mouse pressed)
→ Only the graphic components of the same type chosen for step 2 at the interior of the selection zone appear shining
4

Add another zone to the selection

  • repeat step 3 maintaining the key CTRL pressed to add components to the selection group
→

The user can then consider a manipulation of this selection group (copy/paste, slide, rotate…)

The user can equally consider the Edit array command for the selection consisting in components of the same type

Select all

To select all the electric components present on the sheet:

  • Use the keyboard shortcut Ctrl + A
