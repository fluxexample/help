MaterialCurveSuperimposition

Superimpose material curves

Syntax

MaterialCurveSuperimposition(name, interpolation, materialCurves)

Parameters

  • name : Name of the new 2D curve (material) - [1..1] String

  • interpolation : Interpolation - [1..1] Enumeration

    Potential values :

    • linear
    • spline
  • materialCurves : Material curves to superimpose - [1..N] MaterialCurve

Returns

No return value
